# IPR System Documentation

This folder contains complete technical documentation for the Contract Risk Analyzer system.

## Documents

1. `SYSTEM_OVERVIEW.md`
   High-level product purpose, capabilities, and end-to-end behavior.

2. `ARCHITECTURE.md`
   Logical architecture, data flow, and subsystem boundaries.

3. `WORKFLOWS.md`
   Runtime workflows for upload, analysis, summarization, and chat.

4. `COMPONENTS.md`
   Backend and frontend component inventory with responsibilities.

5. `API_REFERENCE.md`
   REST API endpoints, request/response contracts, and status behavior.

6. `OPERATIONS.md`
   Local runbook, environment configuration, and troubleshooting.

## Reading Order

If you are new to the project, read in this order:

1. `SYSTEM_OVERVIEW.md`
2. `ARCHITECTURE.md`
3. `WORKFLOWS.md`
4. `COMPONENTS.md`
5. `API_REFERENCE.md`
6. `OPERATIONS.md`
